## 🌟 Cos'è StaffSystemReloaded?

**StaffSystemReloaded** è una versione migliorata e aggiornata del classico StaffSystem, progettata per offrire un'esperienza moderna, fluida e completa per lo staff del tuo server.

👉 Disponibile anche su Spigot:
https://www.spigotmc.org/resources/staffsystemreloaded-like-coralmc.127823/

Forkato e migliorato, il plugin introduce funzionalità avanzate, fix importanti e uno stile ispirato ai server moderni come CoralMC.

✔️ Perfetto per server **PvP / Practice / Network**
✔️ Leggero e performante
✔️ Pronto all'uso

---

## 🔥 Funzionalità Principali

* 🛡️ **Staff Mode completa**
* 🔍 **Controllo player avanzato**
* 🧰 **Item staff personalizzati (stile CoralMC)**
* ⚙️ **Messaggi configurabili**
* 🚫 **Gestione permessi intelligente**
* 🔁 **Alias comandi migliorati**

## 🛠️ Requisiti

* Java 8+
* Spigot / Paper / Bungee
* Compatibile dalla **1.8 alla 1.21+**

---

## 📦 Installazione

1. Scarica il plugin da Spigot o GitHub
2. Inserisci il `.jar` in `/plugins`
3. Riavvia il server
4. Configura `config.yml`
